Word search using react

This is a word search written using React.

The React/JSX code is in ./example/app/public/js/word_search.jsx

It currently gets compiled to normal JavaScript by the Babel transformer by adding the type `"text/babel"` to the script tag that loads the file. e.g. <script src="/js/word_search.jsx" type="text/babel"></script>

The React widget is looks for an element with the id of 'word_search' and then binds the React widget into that element. e.g. <div id="word_search"></div>
